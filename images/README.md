### Multilingual Analysis (from multilingual_sentiment.ipynb)

#### 4. multilingual_performance_comparison.png (393 KB)
**5-Panel Multilingual Performance Dashboard**

Comprehensive comparison of VADER vs Transformer models across English and Hindi:
- **Panel 1:** Accuracy comparison by language and model
- **Panel 2:** F1-score comparison
- **Panel 3:** Confusion matrix - VADER on English
- **Panel 4:** Confusion matrix - VADER on Hindi  
- **Panel 5:** Confusion matrix - Transformer on Hindi
- **Panel 6:** Metrics heatmap (Accuracy, F1, Precision, Recall)

**Key Findings:**
- English: 100% accuracy (both VADER and Transformer)
- Hindi: 100% VADER, 87.5% Transformer (zero-shot)
- Demonstrates cross-lingual transfer learning effectiveness

---

#### 5. model_confidence_analysis.png (286 KB)
**4-Panel Confidence Distribution Analysis**

Deep dive into model confidence and prediction accuracy:
- **Panel 1:** VADER compound score distribution
- **Panel 2:** Transformer confidence score distribution
- **Panel 3:** Confidence comparison for correct vs incorrect predictions
- **Panel 4:** Prediction accuracy by confidence quartiles

**Key Insights:**
- High confidence correlates with prediction accuracy
- VADER shows more uniform confidence distribution
- Transformer exhibits higher confidence on successful predictions
- Error analysis reveals confidence patterns in failure modes

## ðŸ”¬ Generation

### English Analysis Images
Generated by `scripts/generate_visualizations.py`:
```bash
python scripts/generate_visualizations.py
```

Source notebook: `notebooks/maincode.ipynb`
- Cell 11 (lines 143-151) - Review length distribution
- Cell 17 (lines 218-264) - Sentiment word clouds
- Cell 32 (lines 566-589) - Top words bar chart

### Multilingual Analysis Images
Generated by `multilingual/multilingual_sentiment.ipynb`:
- Cell 20 (lines 509-603) - Performance comparison dashboard
- Cell 23 (lines 773-840) - Confidence analysis visualization

To regenerate all images:
```bash
# Generate English analysis images
python scripts/generate_visualizations.py

# Generate multilingual images (run in Jupyter)
jupyter notebook multilingual/multilingual_sentiment.ipynb
# Run visualization cells (20, 23)
```
